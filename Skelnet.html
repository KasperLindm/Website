Skelnet SDL Networking

This was a small group group-project that I and two other programmers made. Our focus was on networking the game and we are happy with what we accomplished in 4 weeks.

It was really fun to work with this project because the team was small and we worked closely together with things we wanted to do and improve. During the project I got to practice alot of structure, teamwork, networking and more!

I mostly worked with the networking, audio, particle-system but we worked toghether alot to solve the problems we faced as a group (such as restructiuring the packet-system).

SNParticleEffect* SNParticleSystem::ReuseParticleEffect()
{
	for (int i = 0; i < particleEffects.size(); i++)
	{
		if (!&particleEffects[i])
			continue;

		if (!&particleEffects[i]->isUsed)
		{
			return particleEffects[i];
		}
		else
		{
			return new SNParticleEffect();
		}
	}

	return new SNParticleEffect();
}

inline Vector2 RotateVector(Vector2 vec, float theta)
{
	SNMat3x3 mat =
	{
		cos(theta),	 sin(theta), 0,
		-sin(theta), cos(theta), 0,
		0, 0, 1
	};

	return MultiplyMat3x3Vec2(mat, vec);
}
            
 
inline Vector2 TranslateVector(Vector2 vec, Vector2 translation)
{
	SNMat3x3 mat =
	{
		1, 0, translation.x,
		0, 1, translation.y,
		0, 0, 1
	};

	return MultiplyMat3x3Vec2(mat, vec);
}
