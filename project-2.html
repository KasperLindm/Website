Project Overview

During my time working on hexagroove I've worked on a large amount of different features but my main tasks were centered around porting to Xbox from Nintendo Switch, this meant making an achievement system and making sure that the build versions were compatible etc.

Additionally I set up a CI-system for this project using Jenkins. This simplified and improved the workflow and itteration times for our developers and managers.
